프로젝트 개요



장르: 3D 생존 디펜스

개발 기간: 1개월

엔진: Unity

참여 인원:  6명



프로젝트 소개



Dice Survivor는 웨이브 기반 전투와

주사위 콘셉트 능력 요소를 결합한 생존형 디펜스 게임입니다.





내가 담당한 기능



1\. 웨이브 / 타이머 시스템



10분 생존 구조

시간 경과에 따른 적 강화 구조

보스 스폰 타이밍 제어



2. 랜덤 아이템 드롭 시스템



힐(체력 회복)

코인(수집 재화)

이동속도 증가 버프



3\. 캐릭터 커스터마이징 UI



색상 변경

눈/입/모자 선택

게임 시작 시 선택값 즉시 반영



또한 작업 내용은 Asset/Jang/Scripts 에서 확인 가능합니다.
